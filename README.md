# 🍹 Adega Dose Certa - Catálogo Online

![Status do Projeto](https://img.shields.io/badge/Status-Online-green?style=for-the-badge)
![GitHub Pages](https://img.shields.io/badge/Hospedagem-GitHub%20Pages-blue?style=for-the-badge)

Bem-vindo ao repositório oficial da **Adega Dose Certa**! Este é um Web App moderno, prático e descontraído, desenvolvido para facilitar os pedidos de bebidas e itens de conveniência via WhatsApp.

---

## 🎨 Identidade Visual
O projeto foi inspirado em uma estética **Vintage Party**, utilizando cores vibrantes e um mascote carismático (a caneca animada) para criar uma conexão direta com o público que busca o "esquenta" perfeito ou a resenha com os amigos.

* **Paleta de Cores:** Vermelho, Branco, Amarelo e Preto (Estilo Conveniência Premium).
* **Design:** Mobile-First, otimizado para navegação rápida em smartphones.

---

## 🚀 Funcionalidades Principais

- [x] **Verificação de Idade:** Modal obrigatório para conformidade legal (+18).
- [x] **Catálogo Completo:** Organizado por categorias (Copões, Whiskys, Vodkas, Gin, Cervejas, Vinhos e Tabacaria).
- [x] **Carrinho Dinâmico:** Adição de produtos com soma automática de valores em tempo real.
- [x] **Checkout WhatsApp:** Geração automática de mensagem estruturada com o pedido, endereço e forma de pagamento, enviada diretamente para o WhatsApp da adega.
- [x] **Hospedagem Gratuita:** Site rodando 24h por dia via GitHub Pages.

---

## 🛠️ Tecnologias Utilizadas

O site foi construído utilizando as tecnologias fundamentais da web (Vanilla Stack) para garantir máxima velocidade e compatibilidade:

- **HTML5:** Estruturação semântica do catálogo.
- **CSS3:** Estilização moderna com Flexbox, Grid e variáveis.
- **JavaScript:** Lógica do carrinho de compras e integração com a API do WhatsApp.
- **Google Fonts:** Tipografia 'Bebas Neue' e 'Montserrat'.

## 📦 Como usar este projeto

1. **Acesse o site:** [CLIQUE AQUI PARA VER O SITE AO VIVO](https://markos09y.github.io/adega-dose-certa/)
2. Escolha seus produtos clicando em **"+ Adicionar"**.
3. No rodapé, clique em **"Fazer Pedido"**.
4. Preencha seus dados e pronto! Você será redirecionado para o WhatsApp para finalizar.
